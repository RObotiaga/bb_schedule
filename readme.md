# bb_schedule

Python-скрипт для автоматизации получения расписаний для всех факультетов УрГУПС с сайта bb.usurt.ru.

## Функции:
- Автоматический вход в Blackboard (bb.usurt.ru)
- Переход к разделу "Очная форма обучения" > "Нечетная неделя"
- Автоматический обход всех факультетских папок
- Скачивание всех файлов расписаний в формате .xls в выбранную папку

Инструкция по запуску будет добавлена позже.

## Установка и запуск

1. Установите Python 3.8+
2. Установите зависимости:

```
pip install -r requirements.txt
```

3. Запустите скрипт:

```
python bb_schedule/fetch_schedule.py
```

Можно задать переменные окружения BB_LOGIN и BB_PASSWORD для другого пользователя.

Все скачанные xls-файлы будут сохранены в папку `schedules`.
